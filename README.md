Web-Based Music Player:
 This project is a modern, web-based music player built with HTML, CSS, and JavaScript. It allows users to browse and play a playlist of songs with an interactive   and visually appealing interface.

Key Features:

Playlist Display & Search – Shows song titles, artists, and supports live search to quickly find songs.

Song Playback – Play, pause, and automatically update song info (title, artist, background image, theme color) while the track plays.

Playback Controls – Includes Play/Pause, Next, and Previous buttons for seamless navigation through the playlist.

Dynamic Backgrounds – Changes the player and page background according to the current song for an immersive experience.

Auto-Next Functionality – Automatically plays the next song when the current one ends.

Technologies Used:

HTML – Structure of the player.

CSS – Responsive design, styling, and animated background transitions.

JavaScript – Handles audio playback, playlist management, search functionality, and dynamic UI updates.

User Experience:
The player provides a simple, clean interface with responsive controls, real-time search, and a visually engaging experience through background images and theme colors for each song.
